Design minimalista e luxury, ottimizzato per dispositivi mobili.

## Caratteristiche

- Hero asimmetrico con animazioni in entrata
- Cursore personalizzato oro (desktop)
- Marquee animato con i servizi
- Listino prezzi completo su tre colonne
- Gallery interattiva con hover effect
- Form prenotazione + bottone WhatsApp Business
- Scroll reveal su tutte le sezioni
- Responsive mobile

## Tecnologie

- HTML5 / CSS3 / JavaScript vanilla
- Font: Cormorant Garamond + Jost (Google Fonts)
- Zero dipendenze esterne

## Settore

Progetto dimostrativo per il settore **Beauty & Wellness**.  
Adattabile a: centri estetici, spa, parrucchieri, nail bar.
